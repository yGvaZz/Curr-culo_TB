HTML e CSS

  • Resumo

    Html é uma linguagem de marcação de hipertexto é utilizado para descrever a estrutura de um documento, a estrutura de uma pagina deve conter as seguintes tags elementares

  • Html: É a tag que identifica o documento como um hipertexto html.
  • Head: É a tag que apresenta informacões gerais sobre a página e de configuracão.
  • Title: É a tag que informa o título de uma página na barra do navegador.
  • Body: É a tag que delimita o contéudo da página.

  • O que é tags?


    Como mostrado na imagem acima os elementos HTML tambem chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo crontruida, podendo ser títulos, parágrafos, imagens, links, entre outros.

  • Títulos

    Para os títulos temos ás tags h1 até h6.


    A variação entre as tags definem o grau de destaque do texto.

  • Tags de formatação

    As seguintes tags são utilizadas para a apresentação de conteúdo textual em uma página

  • p: Tag para a representação de um parágrafo.

  • span: Tag para representar um texto sem significado (deve ser usado quando nenhum outro elemento de texto semântico representar um significado adequado).

  • Tags HTML - Listas

    Listas são utilizadas em diversas ocasiões em páginas HTML.

  • A apresentação de itens de um assunto(como do slide explicando o que é tags).

  • Estruturação de novos componetes(como um menu).

    • Exemplo de uma lista desordenada.

  • Tags Link e Nav

    A tag link é utilizada para direcionar o usuário para um novo documento

  • Target: especifica onde abrir o novo documento.

  • Blank: abre em uma nova janela.

  • Self: abre onde o documento atual está (default)

    • Nav é utilizado para representar um agrupamento de links de navegação.
